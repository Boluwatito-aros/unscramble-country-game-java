# Unscramble the Country Game

A geography-based word puzzle game built in Java.

## View the code
The full code is in `Afr_c.java` above.

## About
Developed a Java console game in which players unscramble the names of world countries,
combining language learning with geography in an engaging, timed challenge format.

## Features
- 195 world countries across all continents
- Three difficulty levels based on country name length:
  - Easy (up to 6 characters e.g. Chad, Cuba)
  - Medium (up to 10 characters e.g. Australia)
  - Hard (11+ characters e.g. United Kingdom)
- Progressive hint system — reveals the first letter after 2 wrong guesses, then vowels after 3
- Spaces and hyphens preserved in scrambled names for fairness
- Attempt limit per round with remaining attempts displayed
- Replay option so players can keep playing without restarting

## Tech
Java · OOP · ArrayList · Random · Scanner · Game logic
